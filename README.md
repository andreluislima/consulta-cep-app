<h1 align="center" style="font-weight: bold;">Desafio Técnico - Consulta CEP - Solution SA</h1>
<h3>
      <b>Criar uma aplicação web que permita ao usuário:</b>
</h3>

<p>
    <li>Inserir um <strong>CEP</strong> e buscar automaticamente o endereço utilizando a API <strong>ViaCEP</strong>.</li>
    <li>Preencher dados pessoais (nome e CPF).</li>
    <li>salvar os dados em um banco de dados SQL.</li>
    <li>Gerenciar os dados (listar, editar e salvar alterações).</li>
    <li>Disponibilizar o código em um repositório público no GitHub.</li>
</p>

<h2 id="tecnologias">💻 Tecnologias</h2>

### Frontend
- React
- Axios ou Fetch API

### Backend
- Java

### Banco de Dados
- MySQL, PostgreSQL ou SQLite


### API Externa
- ViaCEP

## Requisitos da Aplicação

### Frontend
- Campo para inserção de **CEP**
- Campos para **Nome e CPF**
- Preenchimento automático de endereço via **ViaCEP**
- Validação de formulário
- Interface responsiva

### Backend
- API REST com operações **CRUD**
- Persistência de dados em banco relacional
